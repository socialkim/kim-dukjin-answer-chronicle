# 김덕진 답변 연대기

> **Built with Codex · Powered by ChatGPT 5.6sol**

같은 AI 질문에 대한 김덕진의 답이 매주 어떻게 달라졌는지 `질문 → 당시 답변 → 변화 요인 → 근거 영상`으로 추적하는 인터랙티브 아카이브입니다.

## 공개 체험

**[김덕진 답변 연대기 바로가기](https://kim-dukjin-answer-chronicle.socialkim.chatgpt.site)**

## 현재 데이터 범위

- 원본 플레이리스트 91편 조사
- 최근 1년 49편, 총 16시간 57분
- 김덕진 출연 47편, 총 16시간 15분
- 10개 진화 질문과 43개 답변 시점
- OpenAI, Google, Anthropic, 중국 AI, AI 경제, 에이전트, 일자리, 반도체, 피지컬 AI, AI 안전

현재의 답변 라벨은 영상 제목과 공개 메타데이터로 만든 분석 시드입니다. 실제 발언 인용으로 단정하지 않으며 UI에서 `AI 분석 · 검토 중`과 `타임스탬프 검수 예정` 상태를 표시합니다.

## 주요 기능

- 자연어 검색과 주제 필터
- 시점별 답변 타임라인
- 직전 답변에서 달라진 점과 변화 사건
- 두 시점 비교
- 원본 YouTube 영상 연결
- 한국어/영어 전환
- 모바일 반응형 UI와 키보드 접근성
- 1년치 전체 영상 메타데이터 CSV

## 로컬 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

검증:

```bash
npm test
```

## 데이터

`data/playlist-2025-2026.csv`에는 최근 1년 전체 49편의 날짜, 영상 ID, 길이, 출연자, 제목이 들어 있습니다.

운영 서비스에서는 채널 소유 자막 또는 Whisper 재전사를 확보한 뒤 질문 군집화, 답변 추출, 변화 비교, 사람 검수 단계를 거쳐야 합니다.
